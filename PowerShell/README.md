### [Manage-WindowsSystem.ps1](https://raw.githubusercontent.com/s0nt3k/Scripts/refs/heads/main/PowerShell/Manage-WindowsSystem.ps1)
A comprehensive Windows system management tool with a GUI interface. Provides common system administration tasks including:
- System information gathering
- Windows Update checking
- Disk cleanup and temporary file management
- Network diagnostics and stack reset
- System File Checker (SFC) execution
- Event log error viewing
- Service management access
- Performance monitoring

Requires PowerShell 5.1 and must be run as Administrator.

### [Invoke-WordCloudFonts.ps1](https://raw.githubusercontent.com/s0nt3k/Scripts/refs/heads/main/PowerShell/Invoke-WordCloudFonts.ps1)
Creates a Word document that lists every available font, rendering each name in its own font so all of the cloud fonts that have not been downloaded yet get downloaded.
