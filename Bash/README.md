# s0nt3k's Bash Shell Scripts

## Available Scripts

### system-deployment-menu.sh
A comprehensive GUI menu system for deploying applications, Docker containers, and managing system administration tasks.

**Features:**
- Deploy Applications & Services (GUI Desktop, Docker, FreePBX, NextCloud)
- Deploy Docker Compose Containers (Pi-Hole, OpenProject, RustDesk, Nginx Proxy Manager, NodeBB)
- OS Administration & Management (Updates, Firewall, Cron Jobs, User Management)
- Built-in About page with MIT license and disclaimer

**Usage:**
```bash
sudo bash system-deployment-menu.sh
```

### setup-menu.sh
WSL-specific setup menu for graphical desktop environment and basic system configuration.

**Usage:**
```bash
sudo bash setup-menu.sh
```
